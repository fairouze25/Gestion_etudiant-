# APPLICATION DE BONJOUR AVEC NODE.JS ET EXPRESS
## C'est une application de bonjour developpée avec nodejs et express. 

## Technologies

- node.js 
- express 

## Structures du projet 
 
Gestion_etudiants/ 
  app.js 
  package.json
  package-lock.json

## Prerequis

- nodejs version 18+

## VErification

- node -v 
- npm -v 

## Installation 


### 1. Cloner le depot 

git clone https://github.com/fairouze25/Gestion_etudiant-.git

### 2. Se deplacer dans le dossier 

cd Gestion_etudiant-.git

### 3. Installation des dependances 

npm install 
### 4. Lancer le projet 

node app.js
